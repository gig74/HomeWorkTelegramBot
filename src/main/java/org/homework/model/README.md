# Модели данных проекта

Этот каталог `src/main/java/org/homework/model` предназначен для хранения классов моделей, которые используются для внутреннего представления данных бота. Классы моделей обеспечивают структурированное представление информации, получаемой из внешних API или других источников, и используются для последующей обработки и представления данных пользователям.

## Назначение классов

### EvilInsultData

Этот класс представляет собой модель данных для JSON-ответа с сайта `evilinsult.com` случайных английских ругательств и используется для его распарсивания. Непосредственно ругательство находится в атрибуте `insult`.

#### TranslatedDataRequest

Этот класс представляет собой модель данных для JSON-запроса сайта переводчика libretrans. В атрибуте `q` находится исходный текст для перевода, в атрибуте `source` - код исходного языка, в атрибуте `target` - код целевого языка перевода.

#### TranslatedDataResponse

Этот класс представляет собой модель данных JSON-ответа с сайта переводчика libretrans. Атрибут `translatedText` используется для переведённого текста.

#### Status

`Status` - перечисление возможных режимов работы бота. Атрибут `text` содержит текстовое описание режима, атрибут `sourceLang` (необяз.) содержит код исходного языка для перевода, атрибут `targetLang` (необяз.) содержит код целевого языка для перевода.
