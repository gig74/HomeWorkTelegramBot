## ConfigBotParameter интерфейс

Файл: `ConfigBotParameter.java`

Интерфейс `ConfigBotParameter` является ключевой абстракцией для получения параметров настройки.

- `String getBotName()` - используется для получения имени зарегистрированного бота.
- `String getBotToken()` - используется для получения токена зарегистрированного бота.

Данный интерфейс используется для обеспечения гибкости и возможности замены источника настроечных параметров без значительных изменений кода приложения.

## ConfigBotParameterImpl класс

Файл: `ConfigBotParameterImpl.java`

Класс `ConfigBotParameterImpl` реализует интерфейс `ConfigBotParameter` и предоставляет конкретную реализацию для получения параметров настройки из переменных ОС.

Аннотация `@Register` указывает на то, что экземпляр этого класса должен быть зарегистрирован в контейнере зависимостей посредством Dependency Injection (внедрение зависимостей), что позволит автоматически внедрить его в любую часть приложения.
